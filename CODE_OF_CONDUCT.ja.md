# Code of Conduct

**日本語** | [English](CODE_OF_CONDUCT.md)

## Our Pledge

私たちは、この project への参加が誰にとっても respectful で constructive な体験になるよう努めます。

## Our Standards

望ましい行動の例:

- 異なる視点や経験を尊重する
- 建設的な feedback を行い、受け入れる
- project と user にとって最善のことに集中する
- mistake に責任を持ち、修正する

許容されない行動の例:

- harassment、threat、discriminatory language
- personal attack または継続的な妨害
- 許可なく private information を公開すること
- professional setting において不適切と合理的に判断される行為

## Enforcement

project maintainer は、この Code of Conduct に反する contribution を削除、編集、拒否できます。また、project に有害と判断される行動に対し、一時的または恒久的に参加を制限できます。

public discussion が不適切な concern は、[SECURITY.ja.md](SECURITY.ja.md) に記載された maintainer contact route に従って private に報告してください。
